# Store Model

Проект "Store Model" — это основа для e-commerce-приложения, реализованная с использованием объектно-ориентированного подхода (ООП).
В рамках проекта описаны основные сущности, такие как продукты и категории, с поддержкой статистики по созданным объектам.

### Возможности 
В рамках проекта реализовано:
- Класс `Product` — описание товаров с поддержкой:
  - приватного атрибута цены `(__price)`, 
  - геттеров и сеттеров с валидацией, 
  - `@classmethod new_product()` для создания продукта из словаря с логикой слияния дубликатов.
  - переопределённый метод `__str__`
  - переопределённый метод `__add__` — сумма стоимости всех товаров `(цена × количество)`  


- Класс `Category` — категории товаров с поддержкой:
  - приватного списка товаров `(__products)`, 
  - метода `add_product()` с автоматическим увеличением счётчика товаров, 
  - геттера products — возвращает отформатированный список товаров.
  - переопределённый `__str__` — строковое представление категории
  - поддержка итерации по продуктам категории с помощью кастомного итератора  


- Загрузка данных из JSON-файла `(load_data_from_json())`, автоматическое создание объектов `Product` и `Category`.

1. Установка:

```git clone https://github.com/BykovMax/store_model.git```

2. Установка зависимостей:

```poetry install```

  - `test_product.py`

- `htmlcov/` - html-отчёт о покрытии

## Добавленные модули 
### модуль `product.py`
- `class Product` - Класс для представления товара в магазине.

### модуль `category.py`
- `class Category` - Класс для представления категории товаров.

### модуль `loaders.py`
- `load_data_from_json` - Загружает категории и товары из JSON-файла, возвращает список Category.
