# Store Model

Проект "Store Model" — это основа для e-commerce-приложения, реализованная с использованием объектно-ориентированного подхода (ООП).
В рамках проекта описаны основные сущности, такие как продукты и категории, с поддержкой статистики по созданным объектам.

### Возможности 
- Описание сущностей `Product` и `Category`
- Подсчёт общего количества категорий и товаров
- Гибкая структура для расширения под реальные задачи e-commerce
- Загрузка категорий и товаров из JSON-файла через функцию `load_data_from_json()`
- Автоматическое создание объектов классов Product и Category из структуры файла

## Установка:
1. Установка:

```git clone https://github.com/BykovMax/store_model.git```

2. Установка зависимостей:

```poetry install```

## Структура проекта

- `src/` - основной код
  - `product.py`
  - `category`
  - `loaders` 
- `tests/` - тесты с использованием `pytest`
  - `conftest.py`
  - `test_category`
  - `test_product.py` 
  - `test_loaders.py` 
- `README.md` - описание проекта
- `htmlcov/` - html-отчёт о покрытии 
- `pyproject.tom` - зависимости и настройки

## Добавленные модули 
### модуль `product.py`
- `class Product` - Класс для представления товара в магазине.

### модуль `category.py`
- `class Category` - Класс для представления категории товаров.

### модуль `loaders.py`
- `load_data_from_json` - Загружает категории и товары из JSON-файла, возвращает список Category.

## Тестирование 
Для тестирования используется `pytest`.

### Запуск тестов 
Перейдите в корневую директорию проекта и выполните:  
```bash
pytest --cov
coverage html
```
